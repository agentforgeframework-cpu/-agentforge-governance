# -agentforge-governance

GitHub URL: https://github.com/agentforgeframework-cpu/-agentforge-governance/blob/main/README.md

Foundational governance standards, principles, and operational guidance for the AgentForge ecosystem.

---

# Purpose

This repository contains governance materials that apply across AgentForge projects, tools, training materials, workflows, and related operational systems.

Governance materials define the principles and standards that help keep AgentForge systems clear, responsible, traceable, and human-centered.

# Folders

## `/governance/`

Canonical governance documents.

This is the authoritative location for governance standards, principles, and related governance files.

## `/documents/`

PDF and document-format support materials.

These may explain, introduce, or provide background for governance documents, but they do not replace the governance documents themselves.

## `/media/`

Visual and media assets that support governance documents.

Images and compiled media support communication and adoption. They are not authoritative replacements for governance text.
