# Purpose

This folder contains official AgentForge starter kits.

Starter kits are canonical starting points that demonstrate how AgentForge governance and kit structure are applied in practice.

They are not separate governance authorities.

They implement governance patterns defined in `/governance/`.

# Current Filename Standard

Starter kits should demonstrate the current AgentForge file standards:

- `SETUP_<kit>.md` for AI-readable setup
- `QUICKSTART_<kit>.md` for human-first startup
- `README.md` in every folder
- lowercase user-facing tool module filenames
- no active `LOAD_*.md` files
- no active plain `QUICKSTART.md` files

---

# Guidance

Starter kits should remain:

- lightweight
- understandable
- operational
- easy to copy
- easy to adapt
- aligned with current governance
