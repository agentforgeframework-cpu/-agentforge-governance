# Purpose

This folder contains official AgentForge starter kits.

Starter kits are canonical starting points that demonstrate how AgentForge governance and kit structure are applied in practice.

They are not separate governance authorities.

They implement governance patterns defined in `/governance/`.

# Guidance

Starter kits should remain:

- lightweight
- understandable
- operational
- easy to copy
- easy to adapt
- aligned with current governance
